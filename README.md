API Authorization (Flask)
Жоба туралы
Бұл жоба Python тілінде Flask фреймворкін қолдана отырып жасалған API авторизация жүйесі. Пайдаланушылар тіркеліп, жүйеге кіріп, қорғалған API сұраныстарын орындай алады.
Мақсаты
- API авторизацияны тәжірибеде қолдану  
- JWT (JSON Web Token) арқылы қауіпсіздік орнату  
- Backend логиканы түсіну  
Қолданылған технологиялар
- Python  
- Flask  
- Flask-JWT-Extended  
- SQLite (немесе басқа дерекқор)  
- REST API  
Авторизация процесі
1. Пайдаланушы тіркеледі (**/register**)  
2. Жүйеге кіреді (**/login**)  
3. Сервер JWT токен береді  
4. Пайдаланушы токен арқылы қорғалған маршруттарға қол жеткізеді
